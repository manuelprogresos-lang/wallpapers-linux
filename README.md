# Manuel Linux Wallpapers

Página estática para publicar fondos de pantalla Linux creados por Manuel / @manuelprogresos.

## Cómo agregar un fondo nuevo

1. Coloca el archivo en `assets/wallpapers/`.
2. Abre `index.html`.
3. Duplica una tarjeta `<article class="card">`.
4. Cambia la imagen, título, descripción y enlace de descarga.

## Publicar gratis con GitHub Pages

1. Crea un repositorio llamado `wallpapers-linux`.
2. Sube estos archivos.
3. Ve a Settings → Pages.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Guarda.

Tu página quedará como:
`https://TU-USUARIO.github.io/wallpapers-linux/`
